## **Умова задачі**

Район має `n` гравців, які пронумеровані від `1` до `n`. Гравці стоять по колу. Гра починається з гравця №1 і рухається за годинниковою стрілкою.

**Правила гри:**
1. Першого гравця (№1) пропускаємо
2. Наступного гравця (№2) видаляємо з кола
3. Продовжуємо: одного пропускаємо, наступного видаляємо
4. Процес повторюється, доки в колі не залишиться один гравець
5. Повернути потрібно **порядок виходу** всіх гравців

##  Приклад

Для `n = 7`:
```
Початковий круг: [1, 2, 3, 4, 5, 6, 7]

Крок 1: Пропускаємо 1, видаляємо 2 → [1, 3, 4, 5, 6, 7]
Крок 2: Пропускаємо 3, видаляємо 4 → [1, 3, 5, 6, 7]  
Крок 3: Пропускаємо 5, видаляємо 6 → [1, 3, 5, 7]
Крок 4: Пропускаємо 7, видаляємо 1 → [3, 5, 7]
Крок 5: Пропускаємо 3, видаляємо 5 → [3, 7]
Крок 6: Пропускаємо 7, видаляємо 3 → [7]
Крок 7: Останній гравець 7

Порядок виходу: [2, 4, 6, 1, 5, 3, 7]
```

## **Інші приклади**

| n | Результат (порядок виходу) |
|---|----------------------------|
| 1 | [1] |
| 2 | [2, 1] |
| 3 | [2, 1, 3] |
| 4 | [2, 4, 3, 1] |
| 5 | [2, 4, 1, 5, 3] |
| 6 | [2, 4, 6, 3, 1, 5] |
| 7 | [2, 4, 6, 1, 5, 3, 7] |
